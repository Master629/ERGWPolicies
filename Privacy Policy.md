# Privacy Policy

**Effective Date:** June 24, 2026

This Privacy Policy explains how the **erlcgiveaways** server utility bot (officially registered as `erlcgiveaways.utils#4743`) handles, processes, and protects your information when you interact with it inside the erlcgiveaways Discord server.

## 1. Data We Collect
To perform its designated utility functions, the bot accesses basic Discord user profile data provided automatically through the Discord API. This includes:
* **Discord User IDs** (Unique numerical identifiers)
* **Usernames and Display Names**
* **Profile Pictures (Avatars)**
* **Rich Presence and Activity Status** (e.g., checking custom status or activity if required by server configurations)

We **do not** collect any sensitive Personally Identifiable Information (PII) such as real names, email addresses, phone numbers, physical addresses, or financial data. 

## 2. How We Use the Data
The data collected is used strictly to ensure the bot functions correctly within the server environment. This includes:
* Processing data submitted through user-initiated custom commands, support systems, or interactive panels.
* Managing administrative server lists and internal data logs required for server operations.
* Executing automated utility tasks as intended by the server management.

We do not sell, rent, or trade your data to third parties. All data remains strictly within the scope of the bot's day-to-day operations.

## 3. Data Retention and Storage
We only keep data for as long as it is actively needed to run the server's internal features, maintain active logs, or fulfill the utility requests of the user or server staff.

## 4. Third-Party Services
Because this bot operates entirely on the Discord platform, your data is also subject to **Discord's Privacy Policy**. We have no control over how Discord handles your data on their platform.

## 5. Your Rights and Data Removal
If you want your data removed from the bot's temporary storage, or if you want to stop the bot from processing your profile information, you can simply leave the erlcgiveaways Discord server or stop interacting with the bot's features. You can also contact a server administrator to request manual data clearing if applicable.

## 6. Updates to This Policy
We may occasionally update this Privacy Policy to reflect updates to the bot's features or changes to Discord's developer guidelines. Continued use of the server bot means you accept these practices.
